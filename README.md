# RustedRover
RustedRover -> локальная осинт утилита с веб интерфейсом, собирающая информацию о пользователе по имеющимся отрывкам информации.
Нацелена на СНГ страны

## Возможности

- Анализ номера телефона
- Поиск упоминаний в соц сетях
- Сопоставление профилей человека
- Безопасное хранение API ключей с AES
- Удобный веб интерфейс
- Поддержка всех Termux, GNU/linux, windows

##Технологический стек
Утилита написана на чистом Rust, позволяя моментально обрабатывать поиск пользователя и искать полноценную информацию о нем.


###Использование
После запуска утилита перейдите в браузере на вебсайт http://127.0.0.1:5354
